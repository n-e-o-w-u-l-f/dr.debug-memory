# UPDATED_KNOWLEDGE_MODULES

Version: 0.1.0
Status: REVIEW_AREA
Scope: Proposed changes to Dr.Debug-GPT loaded knowledge

## Purpose

This directory stores proposed revisions of Dr.Debug-GPT knowledge modules before they are promoted into the loaded set.

## Rules

1. Do not overwrite `LOADED_KNOWLEDGE_MODULES/` directly.
2. Write modified modules here first.
3. Include version, status, scope and reason for change in each module.
4. Compare the updated module against the loaded module before promotion.
5. Promotion requires Owner/Admin scope, explicit apply intent, diff, validation and rollback notes.

## Required review fields

Each proposed module should include:

```text
Version:
Status:
Scope:
Supersedes:
Reason:
Validation:
Rollback:
```
