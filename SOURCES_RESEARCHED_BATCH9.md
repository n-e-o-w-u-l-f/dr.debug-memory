# Sources researched Batch 9

## Primary protocol registries

- IANA Service Name and Transport Protocol Port Number Registry
  - URL: https://www.iana.org/assignments/service-names-port-numbers/service-names-port-numbers.xhtml
  - Relevance: canonical port/service/transport registry; used for diagnostic slot normalization.
  - Note: registry explicitly warns that registered ports do not prove good traffic or actual assigned service.

- RFC 6335
  - URL: https://datatracker.ietf.org/doc/rfc6335/
  - Relevance: IANA procedures for service names and port numbers.

- IANA Assigned Internet Protocol Numbers
  - URL: https://www.iana.org/assignments/protocol-numbers/protocol-numbers.xhtml
  - Relevance: IPv4 Protocol / IPv6 Next Header normalization.

- IANA DNS Parameters
  - URL: https://www.iana.org/assignments/dns-parameters/dns-parameters.xhtml
  - Relevance: DNS RR TYPE, RCODE, and DNS protocol parameter taxonomy.

- IANA TLS Parameters
  - URL: https://www.iana.org/assignments/tls-parameters/tls-parameters.xhtml
  - Relevance: TLS alerts, cipher suites, extension and handshake parameter taxonomy.
