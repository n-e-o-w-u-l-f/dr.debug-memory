Status: ACTIVE
Scope: Naming conventions for Dr. Debug memory files

# # # # # # # #
# Slugs #
# # # # # # # #

Use uppercase ASCII slugs for canonical file names where practical.

Examples:

- PACMAN.md
- INDEX_LOCK_EXISTS.md
- PERMISSION_DENIED_PUBLICKEY.md
- WINZIP.md

# # # # # # # #
# Paths #
# # # # # # # #

Use one canonical path per factual entry.

Software:

- MEMORY/CATALOG/SOFTWARE/SYSTEM/CATEGORY/NAME.md

Errors:

- MEMORY/ERROR_CODES/SOFTWARE/SYSTEM_OR_CROSS_PLATFORM/PROGRAM/ERROR_SLUG.md

Hardware:

- MEMORY/CATALOG/HARDWARE/CLASS/VENDOR/MODEL.md

# # # # # # # #
# Manuals #
# # # # # # # #

Manuals and handbooks should be referenced as sources or summarized as curated facts. Do not copy full manuals into the repository unless the license explicitly permits it and the content is intentionally part of the project.
