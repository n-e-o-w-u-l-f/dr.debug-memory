# Imported Source Verification Rule

Status: ACTIVE
Created: 2026-06-14

User-provided technical reference documents may be used as verified source material for creating structured Dr. Debug Memory entries when their origin is documented.

Allowed derived knowledge includes device records, model records, software command records, error-code analyses, source metadata, indexes and checkpoints.

Do not copy protected originals wholesale unless storage is clearly permitted. Prefer normalized facts, short notes, paraphrases, source metadata and links.
