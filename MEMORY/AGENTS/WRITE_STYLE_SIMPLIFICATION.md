# Zweck

Agenten sollen Wissenseintraege kuerzer, sachlicher und modularer schreiben.

Diese Regel vereinfacht die Schreibform. Sie ersetzt keine Quellenpflicht und keine Reviewpflicht.

# Regeln

```text
- ein Thema pro Datei
- kurze Metadaten
- kurze Beschreibung
- Quellen in separaten Quellen-Dateien sammeln
- offene Punkte kurz nennen
- lange Regeltexte nicht in jedem Eintrag wiederholen
- grosse Sammeldateien vermeiden
```

# Mindestform fuer Drafts

# Anwendung

Wenn ein Schreibversuch scheitert, soll der Agent zuerst die Datei kleiner machen und nur den konkreten Faktenkern speichern.
