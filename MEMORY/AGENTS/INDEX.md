## Read-before-work rule

Before any agent resumes repository work, read:

1. `/AGENTS.md`
2. `/MEMORY/INDEX.md`

## Purpose

This folder is for agent-facing directives. It is not a canonical fact database and must not duplicate manufacturer, device, firmware, driver or source records.

## Duplicate prevention

If an agent directive needs a new data family, create or update the family index first and add a resume note. Do not add the same manufacturer, device, source or fix in multiple places.
