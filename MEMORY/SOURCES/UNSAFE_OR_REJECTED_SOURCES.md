Status: ACTIVE
Scope: Unsafe or rejected source bucket

Sources are rejected when they are unsafe, illegal, unreliable, unverifiable, hostile, or outside the project safety boundaries.

Rejected sources must not be used for canonical factual content.

A rejected source entry should include:

- source id,
- reason for rejection,
- date checked,
- affected topic,
- replacement source if available.
